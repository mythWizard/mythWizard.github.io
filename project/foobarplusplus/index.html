FooBar++

May 25, 2018

FooBar++ is a project I worked on for my Senior Capstone in Computer Science at Longwood University. It is a web-based system for Computer Science 160 and 162 students to test their code for projects and for instructors to assign work. Students will be able to work in a safe, sandbox environment. This system is built around operation on an internal network. It is intended to be run on Linux- or UNIX-based machines.
